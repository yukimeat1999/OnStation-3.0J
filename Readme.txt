【OnStationについて】

〔本家プログラム〕
著作権 (c) 1995-2000-Taco van Ieperen, カナダ
OnStationは、洞窟探検中に収集した調査データを記録するために設計されています。
測定したデータを保存し、論理的に整理するための包括的な機能を備えています。

強力な保存と検索機能に加え、
OnStationは世界トップクラスの描画機能を備えています。
洞窟システムとその周辺の地表を、さまざまな色と描画オプションを使って描くことができます。

OnStationのループクロージャー機能により、
ループ (2つの異なる方法で到達する洞窟のセクション) のエラーを自動的に修正することができます。
これにより、OnStationを使用して、公開地図の出発点としてデータを処理することができます。

URL: https://github.com/ncave/OnStation.git
(GitHubのREADME.mdより引用)

【インストール方法】
　ここではOnStationの導入方法について解説する。

1.デスクトップにzipフォルダを保存する。

2.保存したzipフォルダを選択後、
　右クリックしたときに表示されるショートカットメニューの［すべて展開(T)］をクリックし、
　表示された「圧縮(ZIP 形式)フォルダーの展開」ウィンドウの［展開(E)］を選択し、zipフォルダを展開する。

3.展開した "OnStation" フォルダを "C:\Program Files" 直下にコピー&ペーストし、
　このとき表示される「対象のフォルダーへのアクセスは拒否されました」ウィンドウ上にある［続行(C)］をクリック。

4."C:\Program Files\OnStation" のディレクトリ内にある
　"OnStationSetup.bat" をダブルクリックしてアプリを起動させる。

5."OnStationSetup.bat" が起動するとコマンドプロンプトが起動され、
　次の1文を表示し、ウィンドウが自動で閉じられる。
　前項で起動した "OnStationSetup.bat" を
　［管理者として実行(A)］で起動させると、この画面は表示されない。
　・Since it was executed with general privileges, re-execute it with administrator privileges.

6.前項のコマンドプロンプトに続けて、「RegSvr32」ウィンドウが２度表示され、
　それぞれのウィンドウ上に以下の2文が表示される。
　それぞれのウィンドウ上にある［OK］をクリックし、ウィンドウを閉じる。
　・C:\Program Files\OnStation\grid32.ocx の DllRegisterServer は成功しました。
　・C:\Program Files\OnStation\pdate32.ocx の DllRegisterServer は成功しました。

7."OnStation.exe" を右クリックしたときに表示されるショートカットメニューの
　［ショートカットの作成(S)］をクリックし、
　「ショートカット」ウィンドウ上の［はい(Y)］をクリックすることで、
　デスクトップ上にショートカットが作成される。

8.作成したショートカットを使って "OnStation.exe" を起動する。

9.起動したアプリのウィンドウ上でマウスをスクロールするとアプリが終了する場合は、
　"C:\Program Files\OnStation" のディレクトリ内にある
　下記の２つのDLLファイルを削除すると正常に動作する。
　[Ctrl] キーを押しながら、２つのDLLファイルをマウスで選択し、
　右クリックすると表示されるショートカットメニュー内の［削除(D)］を選択。
　表示された「ファイル アクセスの拒否」ウィンドウにある
　「すべての項目にこれを実行する(A)」にチェックを入れ、
　［続行(Ｃ)］をクリックすると２つのDLLファイルが削除される。
　・GLU32.DLL
　・OPENGL32.DLL
